# Sobre o curso

Dominar os princípios fundamentais da lógica de programação é essencial para se tornar um programador eficiente em qualquer linguagem. Portanto, se você está iniciando sua jornada nessa área, o curso Introdução à Lógica de Programação é para você!

Ao longo das aulas, exploraremos conceitos aplicáveis a praticamente todas as linguagens de programação. Mais do que isso, ajudaremos você a desenvolver uma mentalidade lógica para abordar e resolver problemas computacionais.

Com tópicos que vão desde variáveis e operadores até estruturas de repetição, este curso fornece uma base sólida para todos os alunos da Asimov Academy, mesmo para aqueles que nunca pensaram em programar antes.
